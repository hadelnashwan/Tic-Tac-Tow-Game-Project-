# Tic Tac Toe Game 🎮

A simple Tic Tac Toe (XO) game built using C# and Windows Forms.

## ✨ Features
- Two-player mode
- Interactive UI
- Win detection
- Draw detection
- Reset game button

## 🛠️ Technologies Used
- C#
- Windows Forms

## 🎯 How to Play
1. Run the application.
2. Player X starts first.
3. Players take turns clicking on the grid.
4. The first player to align 3 symbols wins.
5. If all cells are filled with no winner, it's a draw.

## 📂 Project Purpose
This project is created for learning and practicing:
- Event handling in C#
- UI design in Windows Forms
- Basic game logic
